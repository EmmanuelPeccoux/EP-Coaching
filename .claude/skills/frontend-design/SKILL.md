# Frontend Design Skill — EP Coaching Site

Creates distinctive, production-grade frontend interfaces with high design quality.

## When to use
- Build or redesign sections (hero, cards, CTAs, forms, nav)
- Create visual components that need premium aesthetics
- Improve animations, transitions, hover effects
- Refactor CSS for better readability or performance

## EP Coaching Design DNA
- Ultra-dark: background #050505
- Rouge signature: #E60000
- Bebas Neue pour tous les titres
- Inter/Montserrat pour le corps de texte
- Grain overlay SVG pour la texture premium
- Custom cursor avec magnetic buttons
- Parallax sur les images hero
- Split text + scroll reveals pour les animations d'entrée
- Marquee pour l'énergie de marque

## Standards de qualité
- Jamais de design générique ou "AI-looking"
- Chaque section doit avoir une raison d'être dans le tunnel de conversion
- Mobile-first toujours (breakpoints 1100px, 700px)
- Performance : images lazy-loaded, animations GPU-accélérées (transform/opacity)
- Les CTAs sont toujours visibles et clairs
